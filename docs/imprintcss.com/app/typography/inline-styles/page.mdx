# Inline Styles

## Introduction

HTML comes with a number of tags that are designed to be used for styling text inside a paragraph.

## List of Styled Tags

Markup | Looks Like
-------|-----------
`<abbr>`   | <abbr title="abbreviation">This is abbreviated text.</abbr>
`<del>`    | <del>This is deleted text.</del>
`<em>`     | <em>This is emphasised text.</em>
`<ins>`    | <ins>This is inserted text.</ins>
`<kbd>`    | An example key: <kbd>CTRL+F</kbd>
`<mark>`   | <mark>This is highlighed text.</mark>
`<q>`      | <q>This is a short quotation.</q>
`<s>`      | <s>This is strikethrough text.</s>
`<small>`  | <small>This is small text.</small>
`<strong>` | <strong>This is strong text.</strong>
`<sub>`    | <sub>This is subscript text.</sub>
`<sup>`    | <sup>This is superscript text.</sup>
`<time>`   | The UNIX epoch is <time>1973-01-01 00:00:00</time>.
`<u>`      | <u>This is underlined text.</u>
`<var>`    | <var>2</var> x <var>2</var> = 4.
